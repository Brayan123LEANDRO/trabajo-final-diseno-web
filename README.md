# Helados Brayan G

¡Bienvenidos a la heladería más dulce de la ciudad!

Sitio web moderno, responsive y con diseño profesional para la helado artesanal **Helados Brayan G**, hecho con puro cariño y mucho helado de vainilla mientras programaba.

### Visita el sitio en vivo

### Características del proyecto
- Diseño 100 % responsive (se ve perfecto en celular, tablet y PC)
- Menú hamburguesa en móviles
- Modo claro y oscuro con preferencia guardada
- Galería de productos con efectos hover y click
- Página Sobre Nosotros con historia y valores
- Código limpio, semántico y bien comentado
- Fuentes modernas de Google (Fredoka One + Poppins)
- Totalmente funcional sin frameworks (solo HTML, CSS puro y un toque de JavaScript)

### Tecnologías usadas
- HTML5
- CSS3 (Flexbox + Grid + Variables CSS)
- JavaScript vanilla
- Git & GitHub
